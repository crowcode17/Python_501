The ndarray Object

NumPy’s main object is the homogeneous multidimensional array (ndarray). It’s a table of elements of the same type, indexed by a tuple of non-negative integers. In NumPy, dimensions are called axes.

For example, the array [1, 2, 1] has one axis with 3 elements. The 2D array below has 2 axes - the first axis has length 2, the second has length 3:

[[1., 0., 0.],
 [0., 1., 2.]]

Array Creation

There are several ways to create NumPy arrays.

From Python Lists/Tuples

# Create array from a list
a = np.array([2,3,4])
print(a)

# Create array with floats
b = np.array([1.2, 3.4, 5.6])
print(b)
[2 3 4]
[1.2 3.4 5.6]

Multi-dimensional Arrays

# 2D array from nested sequences
b = np.array([(1,2,3), (4,5,6)])

Indexing and Slicing

One-dimensional Arrays

One-dimensional arrays can be indexed and sliced like Python lists:

# Create an array
a = np.arange(10)**3
print(a)

# Basic indexing
print(a[2])

# Slicing
print(a[2:5]) # 2nd, 3rd, 4th, NOT 5th

# Advanced slicing and assignment
print(a[:6:2]) # start stop step - first position blank = start at 0

# Reverse the array
print(a[::-1]) # step backwards
[  0   1   8  27  64 125 216 343 512 729]
8
[ 8 27 64]
[ 0  8 64]
[729 512 343 216 125  64  27   8   1   0]

Multi-dimensional Arrays

Multi-dimensional arrays can have one index per axis:

b = np.arange(15).reshape(5,3)

# Single element access
print(b[1,2])

# Slicing rows and columns
print(b[1:3, :]) # all columns from the first and second row

# When fewer indices are provided, missing indices are complete slices
print(b[0]) # entire 0th row
5

 [6 7 8]]

Array Slicing Practice

              [  5,   6,   7,   8],
              [  9,  10,  11,  12]])

Write the necessary python code to obtain each of the following slices:

# 1: 11
print(A[2,2])

# 2: [1, 5, 9]
print(A[:,0])

# 3: [4, 8, 12]
print(A[:,-1])

# 4: [5, 6, 7, 8]
print(A[1])

# 5: [[1,  3], 
#     [5,  7],
#     [9, 11]]
print(A[:, 0:3:2])

# 6: [[ 2,  4], 
#     [ 6,  8],
#     [10, 12]]

# 7: [ 6  7  8  9 10 11 12]
mask = A >= 6
print(A[mask])

# 8: [1, 3, 5, 7, 9, 11]
mask = A % 2 != 0
print(A[mask])
