Boolean Masking

We can create an array of True/False boolean values using comparative operators such as <, >, and ==. We can then use these boolean arrays to index into our data!

For the runtime variable, create a boolean array called longRuntimeMask that is True for elements that are greater than the mean. Use this boolean array to create a new array called longRuntime that only contains the elements that are greater than the mean.

Then use the len function to print out how many movies are longer than the mean.

# Create a boolean that is True for elements greater than the mean
longRuntimeMask = run > np.mean(run)

# Use the boolean array to get only long runtime rows
longRuntime = run[longRuntimeMask]

# Use the len function to print the number of movies longer than the mean
print("movies longer than the mean:",len(longRuntime))
movies longer than the mean: 273

Now we will apply this boolean mask to a different variable of the same length. Use longRuntimeMask and its inverse (you will need the ~ operator) on the score variable to get the average score for both shorter run times and longer run times.

# Create an array with only the scores for long movies
longMovieScores = scores[longRuntimeMask]

# Create an array with only the scores for short movies
shortMovieScores = scores[~longRuntimeMask]

# Print out the average rating for both long and short movies
print("average rating for long movies:", np.mean(longMovieScores))
print("average rating for short movies:", np.mean(shortMovieScores))
